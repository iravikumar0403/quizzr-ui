# Deep-Quiz

This quiz app will check your knowledge on space and space missions.

## Run Locally

Clone the project

```bash
  git clone https://github.com/ravikumar0403/quizzr-ui.git
```

Go to the project directory

```bash
  cd quizzr-ui
```

Open projects

```bash
    index.html
```

Run code.

## Contains

- Homepage
- Category page
- Rules page
- Questions page
- Result page

## Tech Stack

**Client:** HTML, CSS, Javascript

**Hosted:** Netlify
